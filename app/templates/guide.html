User Guide

Rx Verify provides a drug interaction tool for all users. An account is required to access the Medication page, which is where you can keep track of current and past medications, and access information related to those medications.
When completing the registration form, some patient information will be required to make a profile. After submitting the form, a confirmation email will be sent to the provided email address.
The Medication page contains an Active Medication tab and a Historical Medication tab. The Active Medication tab will contain a list of medications you are currently taking. The Historical Medication tab will contain information you are no longer taking.
When adding an active or historical medication, you will be prompted to enter information pertaining to the medication itself. Required fields include the medication name, the strength and directions. Note that a medication name will be suggested based on what you type - you will need to select one of the suggestions. The same can be said for the medication strength - you will need to select from the suggested list.
Once a medication is listed, you will be able to view the information that was entered. If you click on the Comment column, a pop-up will show you the text you entered. On the right side, each entry will contain icons that allow you to edit and delete entries. For active medications, a third icon will redirect you to a page from the MedlinePlus for additional information.
When editing a medication entry, you will be able to make changes to any of the information you had originally entered. For active medications, there is an additional option to move an entry to historical records. This is for the purpose of medications you are no longer actively taking.
The medication interaction tool consists of a text field to type in the name of the medication. Once you have typed and selected the medication of interest, click on the Add button. For patients with an account, you can click on the checkbox to include active medications without having to type them into the text field. You can also remove medications that you have previously added. Simply click on the medication name and click the Remove button. Clicking on the Submit button will run the interaction tool and display the result. This will also clear the previously added medications.
